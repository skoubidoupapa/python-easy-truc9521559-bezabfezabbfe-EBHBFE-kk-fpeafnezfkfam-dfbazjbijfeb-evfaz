Configuration de la base de données avec UwAmp

1. Télécharger et installer UwAmp

  • Rendez-vous sur le site officiel d'UwAmp.
  • Téléchargez la dernière version stable de UwAmp pour Windows.
  • Exécutez l'installateur et suivez les instructions.

2. Lancer UwAmp

  • Une fois UwAmp installé, ouvrez-le.
  • L'interface d'UwAmp affiche plusieurs informations :
    • Le serveur Apache est indiqué par "Apache Status".
    • Le serveur MySQL est indiqué par "MySQL Status".
  • Cliquez sur "Start" à côté d'Apache et MySQL pour démarrer les services. Une fois démarré, Apache et MySQL devraient afficher "Running".

3. Accéder à phpMyAdmin pour gérer la base de données

UwAmp intègre phpMyAdmin, un outil graphique pour gérer facilement les bases de données MySQL.

  • Ouvrez votre navigateur et allez à l'adresse suivante : http://localhost/phpmyadmin
  • Les identifiants par défaut sont :
    • Utilisateur : root
    • Mot de passe : (vide, aucun mot de passe par défaut)

4. Créer une base de données

  • Cliquez sur Bases de données dans le menu principal de phpMyAdmin.
  • Entrez un nom pour votre nouvelle base, par exemple test_db, puis cliquez sur Créer.

5. Créer des tables dans la base de données

  • Cliquez sur le nom de la base de données dans le menu à gauche.
  • Cliquez sur l'onglet Structure et cliquez sur Créer une table.
  • Entrez un nom pour la table, par exemple utilisateurs, et définissez les colonnes comme suit :
    • id : INT, Auto-incrémenté, Clé primaire
    • nom : VARCHAR(100)
    • email : VARCHAR(100)

6. Insérer des données dans la table

  • Cliquez sur l'onglet Insérer et entrez des données comme suit :
    • nom : "John Doe"
    • email : "john.doe@example.com"
  • Cliquez sur Exécuter pour insérer les données.

7. Se connecter à la base de données depuis PHP

Voici un exemple de code PHP pour se connecter à MySQL et récupérer les données d'une table :


        <?php
        $servername = "localhost";
        $username = "root";
        $password = "";
        $dbname = "test_db"; // le nom de votre base de données

        // Créer la connexion
        $conn = new mysqli($servername, $username, $password, $dbname);

        // Vérifier la connexion
        if ($conn->connect_error) {
            die("Échec de la connexion: " . $conn->connect_error);
        }

        // Exécuter une requête SQL pour récupérer les données
        $sql = "SELECT id, nom, email FROM utilisateurs";
        $result = $conn->query($sql);

        if ($result->num_rows > 0) {
            // Sortir les données de chaque ligne
            while($row = $result->fetch_assoc()) {
                echo "id: " . $row["id"]. " - Nom: " . $row["nom"]. " - Email: " . $row["email"]. "<br>";
            }
        } else {
            echo "0 résultats";
        }

        // Fermer la connexion
        $conn->close();
        ?>

8. Configurer Php.ini (facultatif)

  • Ouvrez le fichier php.ini situé dans le dossier bin\php où UwAmp est installé.
  • Apportez les modifications nécessaires puis redémarrez Apache pour appliquer les changements.

9. Accéder à votre projet via le navigateur

  • Placez votre projet PHP dans le dossier www de UwAmp.
  • Accédez à votre projet en ouvrant un navigateur et en tapant : http://localhost/votre_projet/

10. Sauvegarder et restaurer les bases de données

  • Exporter : Allez dans l'onglet Exporter de phpMyAdmin et sélectionnez le format SQL.
  • Importer : Allez dans l'onglet Importer, choisissez un fichier SQL à importer.

Conclusion

Vous avez maintenant un environnement complet pour créer et gérer des bases de données MySQL avec UwAmp. Si vous avez des questions supplémentaires, n'hésitez pas à demander !
